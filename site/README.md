# 🛍️ House Creation by 3MK — Site E-commerce Vitrine

## 📋 Description
Site vitrine artisanal africain pour sacs, bijoux, babouches, parfums, montres et huiles capillaires.

## 🚀 Déploiement GRATUIT (0 serveur)

### Option 1 — Netlify (le plus simple !)
1. Allez sur https://netlify.com
2. Créez un compte gratuit
3. Glissez-déposez tout le dossier `site/` directement sur Netlify
4. Votre site est en ligne instantanément !
5. Vous obtenez un lien type : `https://house-creation-3mk.netlify.app`

### Option 2 — GitHub Pages
1. Créez un compte sur https://github.com
2. Créez un nouveau repository nommé `house-creation-3mk`
3. Uploadez tous les fichiers
4. Allez dans Settings > Pages > Source: main branch
5. Votre site: `https://votrenom.github.io/house-creation-3mk`

### Option 3 — Vercel
1. Allez sur https://vercel.com
2. Créez un compte
3. Glissez-déposez le dossier
4. Site en ligne !
